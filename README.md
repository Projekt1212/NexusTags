# Features

### Features
`MySQL Cross-Server`
`Unlimited tags`
`Confirmation GUI`
`Sorting Type (Filter Hopper)`
`Placeholders`

### User Commands
`/tag` - Open Tag GUI menu.
`/tag help` - Help Command.

### Admin Commands
`/tag create <tagId>` - Create new tag.
`/tag edit <tagId>` - Edit specific tag.
`/tag delete <tagId>` - Delete specific tag.
`/tag unlock <player> <tagId>` - Unlock specific tag for player.
`/tag remove <player> <tagId>` - Remove specific tag for player.
`/tag reload` - Reload plugin.
`/tag dbmigrate <push/pull>` - Push = Save Data from this server to Database, Pull = Load data from Database.

### Placeholders
`%nexustags_tag%` - Display Tag
`%nexustags_active%` - Active Tag ID
`%nexustags_active_formatted%` - Output replacements "_" to space
`%nexustags_unlocked_tag%` - Owned tag by player
`%nexustags_unlocked_tag_formatted` - Owned tag by player (Formatted)
`%nexustags_rarity_<tag_id>` - Show rarity from tag
`%nexustags_preview _<tag_id>` - Show preview message from tag
`%nexustags_total_tag` - Total all tags in Server
`%nexustags_total_tag_formatted` - Total all tags in Server (Formatted)
